# MBMC RAFFLE 2025

To run the program:

1. Open the terminal and navigate to the correct directory.

```
PS C:\Users\DELL> cd Downloads\MBMC_Raffle_wheel
PS C:\Users\DELL\Downloads\MBMC_Raffle_wheel> npm run dev
```

2. Open the localhost link from vite.

### Excel Setup

The excel file should have header columns named as "Set N" Ex: Set 1, Set 2, Set 3, ...

| Set 1       |   Set 2   |      Set 3 |
| ----------- | :-------: | ---------: |
| John Doe    | Doe John  | Johnny Doe |
| Johnson Doe | Donny Joe |  James Doe |

### Set Selection

To select each set, the keyboard hotkey is (Ctrl + Alt + N).
Ex: Set 3 -> Ctrl + Alt + 3
